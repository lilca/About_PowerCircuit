# 電源回路の構成

入力 ---> 実効電圧の変更 ---> 整流 ---> 平滑 ---> 安定 ---> 出力

## DC-DC昇圧コンバータの場合
1. スイッチングにより矩形波を生成（実効電圧の変更）

    コイルを入力電圧に直列につないで電圧を上げる（？）
    PWMを生成
    上がった入力電圧をスイッチング

2. ダイオードを使って半波（全波）整流
3. コンデンサを使って整流した電圧を平滑化
4. 安定化は勉強中
